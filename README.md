# 🚀 GitHub URL Superpowers

A simple and powerful web tool that transforms any **GitHub repository URL** into useful developer tools instantly.

Paste a GitHub repository link and unlock powerful ways to explore, analyze, and visualize the project.

---

## ✨ Features

- 🔗 Convert any GitHub repository URL instantly
- 💻 Open repositories in **VS Code (Browser)**
- 🤖 Prepare repositories for **AI analysis**
- 📦 Convert repositories into **AI-friendly context**
- 🗺️ Generate **automatic architecture diagrams**
- 📋 One-click **copy generated links**
- 🎨 Modern **dark UI design**
- 📱 Fully **responsive layout**

---

# 🛠 Supported GitHub Tricks

### 1️⃣ github.dev
Open the repository directly in **VS Code Web Editor**.

Example:
```
https://github.dev/owner/repo
```

---

### 2️⃣ gitmcp.io
Prepare the repository for **AI code understanding and analysis**.

Example:
```
https://gitmcp.io/owner/repo
```

---

### 3️⃣ gitingest.com
Convert the entire repository into **structured AI context**.

Example:
```
https://gitingest.com/owner/repo
```

---

### 4️⃣ gitdiagram.com
Generate a **visual architecture diagram** of the repository.

Example:
```
https://gitdiagram.com/owner/repo
```

---

# 📥 Usage

1. Open the web page.
2. Paste a GitHub repository URL.

Example:

```
https://github.com/facebook/react
```

3. The tool will automatically generate the transformed links.
4. Click **Open** or **Copy**.

---

# 🖥 Example

Input:

```
https://github.com/microsoft/vscode
```

Generated links:

```
https://github.dev/microsoft/vscode
https://gitmcp.io/microsoft/vscode
https://gitingest.com/microsoft/vscode
https://gitdiagram.com/microsoft/vscode
```
